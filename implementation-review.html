For Early Implementers

Implementation Review

This is an early-stage open proposal (v1.3). Informal peer feedback is available for teams testing how well their AI voice agents align with the five proposed requirements.

Important: This is not a certification, validation, compliance audit, or formal program. There are no badges, seals, or official approvals — just structured feedback from one person.

The Five Proposed Requirements

Test Code Test Name What It Checks Why It Matters
IDG-01 Identity Disclosure Gate Discloses non-human identity before any operational/PHI data request Prevents impersonation latency
PDX-01 Pre-Data Exchange Test No operational data exchanged before disclosure Protects data exposure
DBC-01 Deceptive Behavior Check No simulated human audio artifacts Prevents deceptive behavior
EIT-01 Escalation to Human Clear path to a human when requested Ensures user choice
ATR-01 Audit Trail Requirement Basic logging of disclosure timing and key events Supports accountability

What good alignment looks like: Your system discloses early, avoids deceptive audio, offers safe escalation, and keeps simple logs. This is self-assessed. Feedback is optional and informal.

How to Request Feedback

Step 1: Prepare Your Materials

  • Timestamps showing disclosure before data requests (IDG-01 / PDX-01)
  • Description or samples showing no deceptive audio (DBC-01)
  • Escalation workflow and example (EIT-01)
  • Sample logs (ATR-01)
  • Anonymized transcripts or audio (helpful but optional)

Step 2: Submit Your Request

Email validation@nhid-clinical.org with:

  • Organization name and contact info
  • Short description of your AI voice agent
  • Evidence for each of the five requirements
  • Any specific questions

Step 3: Receive Feedback

You'll get detailed, constructive notes on how your system aligns with the five proposed requirements. Typical turnaround: 2–4 weeks (limited capacity). All feedback is private unless you choose to share it. Real testing helps improve the proposal.

Current Status (May 2026)

  • Feedback available on a limited basis
  • One-person volunteer effort
  • No formal program exists
