Certification framework

Tiered trust levels for AI voice systems.

NHID-Clinical certification provides L1, L2, and L3 trust levels for organizations implementing disclosure, escalation, audit, and operational evidence controls.

L1 Baseline

Minimum conformance

The system passes all five conformance tests under controlled evaluation conditions.

  • All five CTS tests passed.
  • Internal QA self-attestation.
  • Signed declaration and retained evidence.
L2 Operational

Production evidence

The system maintains conformance in production with structured operational evidence.

  • All L1 requirements.
  • Minimum 30 days production logs.
  • Disclosure Failure Rate below 2%.
L3 Enterprise

Independent verification

Full independent verification by a qualified third-party auditor.

  • All L1 and L2 requirements.
  • Auditor attestation.
  • Annual renewal and registry listing when available.
